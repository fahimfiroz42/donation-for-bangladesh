What is the Document Object Model (DOM)?

DOM is short form of Document Object Model an interface for HTML.HTML is the language that builds websites, and JavaScript is the language that makes the information dynamic on those websites change over time.Therefore, HTML and Javascript must work together to develop and manage a website.

However, Javascript cannot understand HTML elements, which is where DOM comes for help. The Document Object Model (DOM) turns illegible HTML elements into objects, making it easier for Javascript to access the material.

The structure of the DOM is of a tree or a forest. The top element of the structure represents the root node of the html element, and the branches represent all the other elements that come under the html tag.

How do you select an element from the DOM?

One way of selecting an element is by using the .getElementById() method and passing the desired ID as an argument.

Another way of selecting an element is by using the .querySelector() method and passing a CSS selector as an argument.

The .querySelectorAll() method works in a similar way to .querySelector(), but it returns an array-like object of all the elements that match the CSS selector.The object returned from .

Another way of selecting an element is by using the .getElementsByClassName() Selects all elements with the specified class name. There are two more way .getElementsByTagName() and .getElementsByName()

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a DOM-based technique for handling events on a collection of elements by attaching a single event listener to a parent element rather than adding individual event listeners to each child.

How it works

  • We connect a single event listener to a parent element that contains all the elements We are interested in, rather than attaching an event listener to each element individually.
  • When an event occurs on one of the child elements, the event bubbles up the DOM tree and reaches the parent element.
  • The event listener on the parent element checks the target of the event and performs the desired action.

    • Event delegation is usefulness
  • Efficient memory usage
  • Easy maintenance
  • Flexibility
  • Improved code organization


    • By using event delegation, we can handle events on multiple elements with a single event listener, making our code more efficient, maintainable, and flexible.

    How do you manipulate an element's attributes and styles using the DOM?

    For attributes, we can use the getAttribute() and setAttribute() methods to get and set the value of an attribute

    We can also use the removeAttribute() method to delete an attribute.This method is usefull to delete the atribute.

    For styles, we can use the style property to get and set the value of a style. For example, to set the color of an element:
    p.style.color = '#007bff';

    We can also use the classList property to manipulate the class attribute of an element.For example to add a class:
    h1.classList.add('color-primary');

    To remove a class, we can use the remove()
    h1.classList.remove('color-primary');

    To create, add, and remove elements, we can use the createElement() method to create a new element, the append() method to add an element to the DOM.

    To insert an element at a specific position, you can use the insertBefore() method.
